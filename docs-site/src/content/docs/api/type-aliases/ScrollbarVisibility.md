> **ScrollbarVisibility** = `"auto"` \| `"always"` \| `"hover"` \| `"never"`

Defined in: [src/components/layout/ScrollArea/ScrollArea.types.ts:17](https://github.com/SebastianWebdev/entangle-ui/blob/3a4ec9f4fea9c67f55f22d9132b3047d3a5027ac/src/components/layout/ScrollArea/ScrollArea.types.ts#L17)

Scrollbar visibility behavior.
- `auto`: Show on scroll/hover, hide after idle timeout
- `always`: Always visible
- `hover`: Show only when hovering the scroll area
- `never`: Hide scrollbars entirely (still scrollable via wheel/touch)
