> **VectorColorPreset** = `"spatial"` \| `"color"` \| `"none"` \| `"custom"`

Defined in: [src/components/controls/VectorInput/VectorInput.types.ts:27](https://github.com/SebastianWebdev/entangle-ui/blob/3a4ec9f4fea9c67f55f22d9132b3047d3a5027ac/src/components/controls/VectorInput/VectorInput.types.ts#L27)

Color coding preset for axis labels.
- `spatial`: X=red, Y=green, Z=blue, W=purple (standard 3D convention)
- `color`: R=red, G=green, B=blue, A=white
- `none`: all labels use default text color
- `custom`: use `axisColors` prop
