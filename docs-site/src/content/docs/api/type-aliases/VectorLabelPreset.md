> **VectorLabelPreset** = `"xyz"` \| `"rgba"` \| `"uvw"` \| `"custom"`

Defined in: [src/components/controls/VectorInput/VectorInput.types.ts:18](https://github.com/SebastianWebdev/entangle-ui/blob/3a4ec9f4fea9c67f55f22d9132b3047d3a5027ac/src/components/controls/VectorInput/VectorInput.types.ts#L18)

Axis label preset configurations.
- `xyz`: X, Y, Z, W — position, rotation, scale
- `rgba`: R, G, B, A — color channels
- `uvw`: U, V, W — texture coordinates
- `custom`: use `axisLabels` prop
