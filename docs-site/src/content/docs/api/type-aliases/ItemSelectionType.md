> **ItemSelectionType** = `"radio"` \| `"checkbox"` \| `"none"`

Defined in: [src/components/navigation/Menu/Menu.types.ts:39](https://github.com/SebastianWebdev/entangle-ui/blob/3a4ec9f4fea9c67f55f22d9132b3047d3a5027ac/src/components/navigation/Menu/Menu.types.ts#L39)

Type of item selection behavior within a group.
- `radio`: Single selection within group
- `checkbox`: Multiple selection within group
- `none`: No selection state, just click handlers
