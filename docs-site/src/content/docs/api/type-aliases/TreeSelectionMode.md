> **TreeSelectionMode** = `"single"` \| `"multiple"` \| `"none"`

Defined in: [src/components/controls/TreeView/TreeView.types.ts:13](https://github.com/SebastianWebdev/entangle-ui/blob/3a4ec9f4fea9c67f55f22d9132b3047d3a5027ac/src/components/controls/TreeView/TreeView.types.ts#L13)

Selection mode for the tree.
- `single`: Only one node can be selected
- `multiple`: Multiple nodes via Ctrl+Click, Shift+Click
- `none`: No selection, tree is display-only
