> **ChatMessageRole** = `"user"` \| `"assistant"` \| `"system"` \| `"tool"`

Defined in: [src/components/editor/ChatPanel/ChatPanel.types.ts:14](https://github.com/SebastianWebdev/entangle-ui/blob/3a4ec9f4fea9c67f55f22d9132b3047d3a5027ac/src/components/editor/ChatPanel/ChatPanel.types.ts#L14)

Standard message roles in an LLM conversation.
- `user`: Human input
- `assistant`: LLM response
- `system`: Informational/status messages
- `tool`: Tool call invocation and result
